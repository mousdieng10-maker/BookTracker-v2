BookTracker V2

Booktracker/manager is a python project that is meant to be able to keep books that the user has 
manually entered or selected from an open library api.

warning: This project was done by someone learning python and thus might not fit standards.
## Features
- API to search books online and save book details to library.
- Manually enter books into library.
- View library.
- Random book recommendations.


## Built With
- Python
- [Open Library API](https://openlibrary.org/developers/api)
